服務流程

我們以專業的每一步，為客戶確保兩件事情： 一個愉快的購買體驗，和整個過程中都安心放心。

何時送貨？
•
一般在14天內送貨，實際交付時間取決於庫存情況，我們在您訂購時會和您確認。
•
度身訂造的特別項目製作較需時，請個別咨詢。



送貨費用多少？
如購買超過3,000元可享受免費送貨到您的專用位置（車邊交收）。

購買低於3,000元的送貨費由50元起，視乎地點而定。

如果您需要送貨上樓，請向我們的同事查詢具體的費用。



  1. 如何安排送貨？

    我們的同事將與您的設計師或承包商進行協調，您可放心，讓我們代勞。
